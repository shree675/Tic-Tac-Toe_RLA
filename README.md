## Setup and Installation

- Install Python from https://www.python.org/downloads/release/python-380/.
- Run the following:

  ```console
  pip3 install numpy
  pip3 install scipy
  pip3 install pygame
  ```

- In each of the folders in the project, run **main.py** to execute the program:

  ```console
  python main.py
  ```
